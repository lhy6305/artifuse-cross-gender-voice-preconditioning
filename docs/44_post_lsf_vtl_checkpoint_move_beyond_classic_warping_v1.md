# 表示层主线断点：停止继续抠经典 Warp

## 断点目的

这份断点用于防止后续推进时，又重新回到：

- `LSF` 参数小修小补
- `VTL warp ratio / band / smooth` 的继续细调

当前需要固定的是一个更高层判断：

**`LSF` 和 `VTL` 两条经典可解释 warp 路线，到当前为止都没有形成足够正证据。**

## 当前阶段结论

截至当前断点：

- `LSF v1`：整体偏弱，且部分样本开始出现伪影
- `VTL v1`：部分样本可辨，但局部性强，伴随对齐伪影与双声感
- `VTL v2`：比 `v1` 更常可辨，但仍然偏弱，且 `slight artifact` 频率偏高

因此当前不能再把问题理解成：

- `warp_ratio` 还差一点
- `active_until_hz` 还没调准
- `energy_rebalance_ratio` 再拧一版就会好

## 强约束

从现在开始，默认不再继续以下动作：

1. 不继续出 `LSF v2 / v3`
2. 不继续出 `VTL v3 / v4`
3. 不再回到 `pole edit / cepstral delta / classic tract warp` 家族内继续抠局部参数

## 下一步原则

下一步应满足至少一条：

1. 表示层升级到更高层，而不是继续停留在经典手工 warp
2. 方向映射不再只靠固定全局几何变形，而要引入更强的条件化表示
3. 如果继续保留可解释性，也应换到新的表示对象，而不是继续沿当前 `LSF / VTL` 参数化

## 接班口令

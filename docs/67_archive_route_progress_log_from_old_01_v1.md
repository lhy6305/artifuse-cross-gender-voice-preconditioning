# 归档：旧版 01 的路线推进日志摘出

## 说明

本文件从旧版 `docs/01_project_overview_and_plan.md` 中拆出，保留当时混在总览里的长时间序实验推进记录。

旧版 01 中实际混入了大量本应独立成文的内容，包括：

- 阶段推进日志
- 听审结果摘要
- pivot 结论
- machine-only sweep 结果
- 某一轮实验后的下一步决定

这些内容应当使用单独编号文档记录，而不是继续堆在活跃总览中。

## 旧版混入的典型内容

1. `speech-first` 到 `envelope / resonance / formant` 家族的尝试记录
2. `WORLD-guided STFT delta`、`VTL`、`conditional transport`、`low-rank`、`neural latent` 等路线的阶段性结论
3. `LSF v1 -> v7` 的阶段推进摘要
4. machine-first gate 建立后的 sweep 结果与人审流转记录
5. 某一天内连续多轮实验的接班口令

## 当前规则

- 活跃总览不再承载这种按时间堆积的实验日志
- 一次实验记录，新增一个编号文档
- 一个明确 checkpoint，新增一个编号文档
- 只有全局仍然成立的重大决策，才允许回写到活跃总览

## 备注

旧版 01 里这类内容非常多，本归档文档只说明拆分边界，不再把全部历史原文再次复制进来。需要追溯旧版长日志时，按 Git 历史查看旧版 `docs/01_project_overview_and_plan.md` 即可。

## 分支建议
- 如果后续开始多人或多任务并行，优先按任务边界拆分，而不是按文件类型拆分。
- 建议优先出现的分支方向：
  - `task/environment-freeze`
  - `task/data-inventory`
  - `task/analysis-bootstrap`
- 只有在出现真实分支后，才在这里登记“分支名 -> 任务 -> 涉及文件”的映射。
